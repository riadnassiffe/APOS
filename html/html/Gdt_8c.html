struct gdt_entry gdt[5]

Estrutura que guarda a GDT.

Atenção:
Toda GDT deve no minimo ter 3 entradas, uma que sera usada pelo pelo processador, outra para guardar o seguimento da memória onde esta o código do programa e a ultima para guardar para guardar segmineto de Dados. No nosso caso usaremos mais duas entradas na lista, uma para guardar a Tss do ecalonador de processos e outra para a Tss dos processos.
