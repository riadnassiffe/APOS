Descrição Detalhada

Este arquivo tem a finalidade de implementar a parte de mais baixo nível do sistema de arquivo do APOS. Esse sistema de arquivo segue os princípios de um sistema de arquivo FAT.
Autor:
Ríad Mattos Nassiffe.
