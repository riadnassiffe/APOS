void irq_handler ( struct regs *  r  ) 

-> Handlers de interrupcao devem sinalizar o termino da operacao para o chip. Caso nao seja dado o aviso, o sistema nao ira enviar mais sinais.

Veja em: [1] Handlers for IRQs must issue End Of Interrupt (EOI) to one or both PIC chips. For IRQ 0-7: Out_Byte(0x20, 0x20); For IRQ 8-15, both 8259 chips must get EOI: Out_Byte(0xA0, 0x20); Out_Byte(0x20, 0x20);
